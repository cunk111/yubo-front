Front-end Application for node/react+redux boilerplate

## Prerequisite
* git (tested on v2.21.0)
* npm (tested on 6.9.0)
* create-react-app (tested on v2.1.8)
* (alternatively to npm) yarn (tested on v1.13)

## Installation

```
$ mkdir tmp && cd $_
$ git clone git@github.com:cunk111/yubo-front.git
$ cd yubo-front
$ yarn install
```
